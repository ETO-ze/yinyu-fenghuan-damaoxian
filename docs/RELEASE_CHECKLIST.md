# 发布检查清单

## 版本命名

| 版本 | 目标 |
|---|---|
| v0.1-demo | 第一关完整可玩 |
| v0.2-architecture | 关卡数据化与 LevelLoader |
| v0.3-save | 存档、继续冒险、检查点 |
| v0.4-feel | 手感优化、音效入口 |
| v0.5-level2 | 第二关与新机制 |
| v1.0-demo | 3 关可发布 Demo |

## 发布前必须检查

- [ ] README 操作说明与实际一致。
- [ ] project.godot 的主场景正确。
- [ ] 开始界面可以进入游戏。
- [ ] 继续冒险有存档时能恢复。
- [ ] 无存档时继续冒险不会崩溃。
- [ ] 第一关可以完整通关。
- [ ] 少于 5 根羽毛不能通关。
- [ ] 5 根及以上羽毛到达传送门可以通关。
- [ ] 生命归零显示失败 UI。
- [ ] 宝箱只能领奖一次。
- [ ] 音频资源缺失时不崩溃。
- [ ] Godot 输出无红色错误。
- [ ] Windows 导出能运行。
- [ ] Web 导出能进入开始界面。

## Windows 导出建议

输出目录：

```text
builds/windows/
```

文件命名：

```text
yinyu-fenghuan-v0.1-demo-windows.zip
```

检查：

- [ ] exe 能启动。
- [ ] pck 文件与 exe 在同一目录。
- [ ] 中文标题显示正常。
- [ ] 480x854 竖屏比例正常。

## Web 导出建议

输出目录：

```text
builds/web/
```

文件命名：

```text
yinyu-fenghuan-v0.1-demo-web.zip
```

检查：

- [ ] index.html 能加载。
- [ ] WASM 文件路径正确。
- [ ] 浏览器中点击开始游戏正常。
- [ ] 键盘输入正常。
- [ ] 音频自动播放限制不会阻塞游戏流程。

## GitHub Release 内容模板

```markdown
# 银羽风环大冒险 v0.1-demo

## 内容

- 第一关：风之高塔。
- 开始界面。
- 玩家移动、跳跃、滑翔、疾跑、下蹲。
- 羽毛/金币收集。
- 传送门通关。
- 失败与胜利 UI。
- 像素风角色、平台、背景、NPC、宝箱和粒子资源。

## 操作

- A / ←：向左移动
- D / →：向右移动
- Space / W / ↑：跳跃
- 下落时按住 Space：滑翔
- Shift / J + 方向键：疾跑
- S / ↓：下蹲

## 已知问题

- 音效和 BGM 可能仍为占位。
- 第二关尚未加入。
- 设置界面暂时只显示操作提示。
```
