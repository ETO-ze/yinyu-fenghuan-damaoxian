# 第二关测试清单：云桥花园 · 浮云试炼

## 1. 关卡进入

### 1.1 第一关通关后进入第二关

步骤：

1. 开始新游戏。
2. 完成 level_01。
3. 按提示进入下一关。

预期结果：

- 进入 level_02。
- HUD 显示“云桥花园 · 浮云试炼”。
- 玩家出现在第二关起点。

### 1.2 直接加载第二关

步骤：

1. 临时让 `GameSession.current_level_id = "level_02"`。
2. 运行 `Main.tscn`。

预期结果：

- Main 能加载 `Level02Data`。
- 平台、羽毛、金币、传送门生成。
- 无红色错误。

## 2. 云平台机制

### 2.1 云平台可站立

步骤：跳到 `cloud_a`。

预期结果：玩家能站在云平台上，不会立刻穿透。

### 2.2 云平台闪烁

步骤：玩家站上云平台后等待约 0.8 秒。

预期结果：云平台开始闪烁，玩家仍能短暂站立。

### 2.3 云平台消失

步骤：玩家站上云平台后等待约 1.4 秒。

预期结果：云平台隐藏，碰撞关闭，玩家会下落。

### 2.4 云平台恢复

步骤：云平台消失后再等待约 3 秒。

预期结果：云平台重新出现，碰撞恢复，玩家可以再次站上去。

## 3. 第二关通关

### 3.1 羽毛不足时不能通关

步骤：

1. 收集少于 6 根羽毛。
2. 触碰终点传送门。

预期结果：不通关，HUD 提示还需要羽毛。

### 3.2 羽毛足够时通关

步骤：

1. 收集至少 6 根羽毛。
2. 触碰终点传送门。

预期结果：

- 显示胜利 UI。
- 如果 `next_level_id` 为空，提示 Demo 已完成。
- 玩家输入锁定。

## 4. 检查点与失败

### 4.1 中段检查点

步骤：

1. 到达 `garden_mid` 检查点。
2. 故意掉落。

预期结果：

- 扣 1 条生命。
- 玩家回到 `garden_mid` 附近。

### 4.2 生命归零

步骤：连续掉落直到生命归零。

预期结果：

- 显示失败 UI。
- 玩家输入锁定。

## 5. 回归测试

- [ ] level_01 开始位置正常。
- [ ] level_01 普通平台不变成云平台。
- [ ] level_01 羽毛需求仍是 5。
- [ ] level_01 传送门仍在右侧终点。
- [ ] level_01 通关后能进入 level_02。
- [ ] 无 Godot 红色错误。
